悬浮注释 hint

npm install hexo-tag-hint --save

这里是正文

下拉抽屉

{% enddetails %}

文字遮盖效果 Spoiler

npm install hexo-spoiler --save

站点根目录配置填写

spoiler:
  style: blur # 或者box
  color: black # 仅当 style 为 box 时起效
  p: false # 没懂啥意思，不管它

{% spoiler option:value text... %}

{% spoiler 默认配置效果 %} → 默认配置效果

动态终端代码演示

npm install @heowc/hexo-tag-gdemo
